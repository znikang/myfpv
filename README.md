[ExpressLRS-Configurator](https://github.com/ExpressLRS/ExpressLRS-Configurator/releases/)

[https://github.com/ExpressLRS/ExpressLRS](https://github.com/ExpressLRS/ExpressLRS)

當你要刷fireware 時要播一下這開關 搞定後就可以刷刷樂了
